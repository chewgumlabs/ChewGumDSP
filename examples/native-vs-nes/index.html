Native Triangle vs. NES-Style Triangle

Both buttons trigger the same bell synthesis through the same envelope, harmonic structure, and bandpass filter. The only difference is the triangle wave source.

  • Native uses the browser's built-in oscillator.type = 'triangle'.
  • NES-style uses a Web Audio PeriodicWave built by createNesTriangleWave from a 32-step NES-style source table, projected onto 24 cosine/sine harmonics with the DC offset removed.

Audio dormant. Click either button to arm and play.

Deployed contrast

  • Dead Beat and Falling Hall ship the NES-style PeriodicWave.
  • Phosphor uses the browser's native triangle oscillator. Phosphor is the contrast point here, not a deployment of this library.
